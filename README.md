# All-in-One PDF Tool

**Status:** ✅ Working  

All-in-One PDF Tool is a Node.js application that allows users to perform various PDF operations seamlessly. It uses **EJS** for dynamic frontend rendering and **pdf-library** for PDF manipulation. This tool is perfect for creating, editing, merging, and adding page numbers to PDFs with ease.

---

## Features

- Upload and manage PDF files  
- Merge multiple PDFs into one  
- Add page numbers to PDFs  
- Download modified PDFs  
- Simple and user-friendly interface  

---

## Technologies Used

- **Node.js** – Backend runtime  
- **EJS** – Template engine for rendering dynamic pages  
- **pdf-library** – For PDF generation and manipulation
